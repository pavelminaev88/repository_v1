# Personal CV Website - Pavel Minaev

## Overview
A clean, minimalist one-page personal website/CV for Pavel Minaev, a Transformation & Program Leader. The site follows a professional, calm design aesthetic with clear hierarchy and mobile responsiveness.

## Project Structure
- `index.html` - Main HTML page with all CV content
- `styles.css` - CSS styling (clean, modern, responsive)
- `server.py` - Simple Python HTTP server for development
- `CV_web.md` - Original specification document

## Technology Stack
- Static HTML/CSS website
- Python http.server for local development
- No build process required

## Running the Project
The website runs on port 5000 using Python's built-in HTTP server:
```
python server.py
```

## Deployment
This is a static site - deploy using the static deployment option with `.` as the public directory.

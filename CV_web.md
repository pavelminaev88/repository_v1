## 4) Visual & Interaction Spec (Plain Language)

**Layout & Spacing**
- Single column, generous vertical spacing  
- Clear hierarchy, no dense blocks

**Typography Feel**
- Clean, modern, neutral  
- Comfortable reading, not stylized

**Color Palette**
- Light background (white / off-white)  
- One restrained accent color for links and CTAs

**Buttons & Links**
- Text-first links  
- Subtle hover feedback, no animation noise

**Motion Principles**
- Light, natural scroll behavior  
- Minimal reveal or fade-in only if it adds calmness

**Mobile Behavior**
- Fully readable without zoom  
- Same hierarchy, stacked layout  
- CTAs always reachable
